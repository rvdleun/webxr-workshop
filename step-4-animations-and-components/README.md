# Step 4: Animations and Components

## Instructions

* Add the animation component to an entity

## Example

```html
<a-sphere animation="property: rotation; from: 0 0 0; to: 0 360 0; loop: true; easing: linear” position="0 1.6 -3"></a-sphere>
```
